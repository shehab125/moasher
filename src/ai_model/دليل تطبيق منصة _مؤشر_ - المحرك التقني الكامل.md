## 1. نظرة عامة على المشروع

منصة **"مؤشر"** هي أداة ذكية لتقييم وتوقع أسعار العقارات في مدينة جدة. تستخدم المنصة نماذج الذكاء الاصطناعي المتقدمة لتقديم:
*   **تقييم فوري:** حساب سعر العقار الحالي بناءً على الحي والمساحة والنوع.
*   **توقع مستقبلي:** التنبؤ بسعر العقار بعد سنة من الآن.
*   **تحليلات السوق:** إحصائيات شاملة عن أسعار جدة وتوزيع الصفقات.

## 6. الربط مع الموقع (Frontend Integration)

### 6.1 مثال باستخدام JavaScript/React

```javascript
// استدعاء الـ API للحصول على قائمة الأحياء
async function getNeighborhoods() {
  const response = await fetch('http://localhost:8000/neighborhoods');
  const data = await response.json();
  return data.neighborhoods;
}

// استدعاء الـ API لتقييم العقار
async function valuateProperty(neighborhood, propertyType, area) {
  const response = await fetch('http://localhost:8000/valuate', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      neighborhood: neighborhood,
      property_type: propertyType,
      area: area
    })
  });
  const data = await response.json();
  return data;
}

// مثال على الاستخدام
const result = await valuateProperty('الياقوت', 'قطعة أرض', 500);
console.log(`السعر الحالي: ${result.total_current_price} ريال`);
console.log(`السعر المتوقع بعد سنة: ${result.total_predicted_price_1year} ريال`);
```

### 6.2 مثال باستخدام Python/Requests

```python
import requests

# استدعاء الـ API
url = "http://localhost:8000/valuate"
payload = {
    "neighborhood": "الياقوت",
    "property_type": "قطعة أرض",
    "area": 500
}

response = requests.post(url, json=payload)
result = response.json()

print(f"السعر الحالي: {result['total_current_price']} ريال")
print(f"السعر المتوقع بعد سنة: {result['total_predicted_price_1year']} ريال")
```

## 8. استكشاف الأخطاء

### المشكلة: "الحي غير موجود"
**الحل:** تأكد من تهجئة اسم الحي بشكل صحيح. استخدم `/neighborhoods` للحصول على القائمة الكاملة.

### المشكلة: "خطأ في الاتصال بالـ API"
**الحل:** تأكد من أن الـ API يعمل على `http://localhost:8000` وأن الخادم قيد التشغيل.

### المشكلة: "النموذج لم يتم تدريبه"
**الحل:** تأكد من تشغيل `train_muasher_model.py` أولاً قبل تشغيل الـ API.
